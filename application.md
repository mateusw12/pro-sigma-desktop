Quero que seja um especialista em python, e me ajudasse a fazer uma aplicação python desktop bonita e harmonioza para analises estatistica para metogologia Six Sigma, o nome do projeto é Pro Sigma.

Funcionalidades:
- Usuário acessa o software atraves de uma licença que ele irá colocar no sistema, essa licença seria um hash com as seguintes propriedades dentro:
plan, expiratedDate, com isso essa licença vai validar se ele tem acesso ou não ao sistema e quais ferramentas tem acesso.

- O usuário poderá anexar arquivos excel ou csv para fazer as análises de dados
- Teremos histórico salvo de análises (iremos salvar as analises no proprio computador da pessoa)
- Teremos uma pasta no computador do usuário que salvaremos analises.
- As análises teriamos ferramentas conforme abaixo:

- como vamos usar o python não sei como seria por conta das bibliotecas que iremos usar numpy, bibliotecas de gráficos e tantas outras para essas

#### Plano Básico

1. **Variability** - Análise de variabilidade de dados
2. **Process Capability** - Cálculo de Cp, Cpk, Pp, Ppk
3. **Hypothesis Test** - Testes T, Z, ANOVA, Qui-quadrado
4. **Distribution Test** - Ajuste de distribuições (Normal, Weibull, etc)
5. **COV EMS** - Análise de coeficiente de variação
6. **Análise de Distribuição** - Analisar curvas de distribuição dos dados
7. **Analytics** - Fazer analises na proprioa tabela de dados, como regras, formtação das colunas essas coisas simples, criação de colunas calculadas

#### Plano Intermediário

1. **Text Analysis** - Análise textual e frequência de palavras
2. **Normalization Test** - Testes de normalidade (Shapiro-Wilk, KS, etc)
3. **Control Charts** - Cartas X-bar, R, S, P, NP, C, U
4. **Dashboard** - Visualização de métricas
5. **Monte Carlo** - Simulações Monte Carlo

#### Plano Pro

1. **Simple Regression** - Regressão linear simples
2. **Multiple Regression** - Regressão linear múltipla
3. **Multivariate** - PCA, Análise Fatorial, Cluster
4. **StackUp** - Análise de tolerâncias 2D
5. **DOE** - Design of Experiments (Fatorial, Fracionado e 3 niveis)
6. **Space Filling** - Latin Hypercube
7. **Warranty Costs** - Análise de custos de garantia
